FeedCrawler version 1.0
--------
README
--------
@author Nattapol Kemavaha

This application will spawn a crawler to fetch a basic information (title, description
and vendor) from nexTag.com based on the keyword supplied by the user.

---------------
 How to use
---------------
1.) Open a console or command line
2.) Type "java -jar Assignment.jar <keyword> <page>
where <keywords> : your desired keyword or product you'd like to search for
	  <page> [optional]: indicate page number where the products are listed
